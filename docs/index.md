# Juegos de mesa

Esta web recopila distintos juegos de mesa, con las instrucciones resumidas de cómo jugar a cada uno.

En el menú lateral podrás ver los juegos y hacer clic en cada uno. Habrá una presentación con diapositivas de los aspectos fundamentales del juego y, en el caso de que sea necesario, algún apartado más con preguntas frecuentes o cuestiones adicionales.
